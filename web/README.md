# vite-react-ts-admin

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### antd5
```
antd5已经放弃less，使用css-in-js的方式注入样式，
所以安装antd之后不用引入样式都可用
```

### less
```
vite4中已经默认安装less使用，但是建立样式文件必须是.module.less才能使用
```
